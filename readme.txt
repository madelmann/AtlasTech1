Game Object Format:
-------------------

- Model, file to load
- Texture
- Shader
- Transformation
- Scale
- Static (bool), determines if mesh should be stored in a display list



Methods:
--------

Animation:
- nextAnimation
- previousAnimation
- selectAnimation
- stop
- start





Console commands:

spawn object <name>		load and insert an object into the game
spawn monster <name>		load and insert an ai controlled object(=monster) into the game
